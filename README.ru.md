## ✨ Ключевые возможности

- **Agentic ReAct Workflow** — Думает цепочками: `Мысль → Действие (Инструмент) → Наблюдение`. Достигает цели автономно.
- **3-уровневая Само-Эволюция** — Личные коррекции → Мутации генов → Переписывание корневого промпта. Агент улучшается сам.
- **Дистилляция Памяти** — После 100 событий автоматически архивирует логи, предотвращая переполнение памяти телефона.
- **Синхронизация с Целями** — Каждая новая заметка проверяется на соответствие вашим `goals.json`. Нерелевантная информация отсеивается.
- **Дашборд Эволюции** — Напишите `покажи дашборд` для интерактивного дерева мутаций на Mermaid.js и статистической панели.
- **Безопасный Обмен Капсулами** — Экспортируйте знания (без личных данных), импортируйте от комьюнити с защитой от инъекций.

## 🚀 Установка

### Вариант 1: Ручной импорт по URL (Рекомендуется)
1. Откройте **Google AI Edge Gallery** на вашем Android/iOS устройстве.
2. Нажмите **Agent Skills** → **Add Custom Skill** → **From URL**.
3. Вставьте этот адрес:
```
https://github.com/uussnn/second-brain
```
4. Подтвердите. Приложение загрузит и активирует **Второй Мозг v3.3.0**.

### Вариант 2: Клонировать локально
```bash
git clone https://github.com/uussnn/second-brain.git
```
Укажите AI Edge Gallery путь к локальной папке (см. документацию приложения).

## ⚡ Основные команды

Попробуйте эти команды после активации навыка для взаимодействия с агентом:

**📊 Обзоры и планирование**
- `«обзор дня»` / `«обзор недели»`
- `«что у меня сегодня»`

**🧠 Zettelkasten & Заметки**
- `«MOC review»` / `«обнови MOCs»`
- `«покажи MOC [тема]»`
- `«сохрани в Second Brain»` (для сохранения мыслей или фото)

**🗂 Память и Настройки**
- `«что я говорил про [X]»`
- `«покажи мои предпочтения»`
- `«добавить цель [ваша цель]»`

**🧬 Эволюция и Обслуживание**
- `«покажи дашборд»` (интерактивный Evolution Map)
- `«личный review»` (агент анализирует свои правила)
- `«сжать память»` (дистилляция логов)
- `«экспорт Capsule»` / `«импорт Capsule»`

**📧 Gog Workspace**
- `«разбери почту»`
- `«что у меня в календаре»`

## 🗂 Структура репозитория

```text
second-brain/
├── SKILL.md                      # Главный промпт агента (v3.3.0)
├── README.md                     # Английская версия
├── README.ru.md                  # Этот файл
└── Evolution/                    # Начальные заготовки данных (предзаполненные)
    ├── genes.json                # Параметры поведения агента (гены)
    ├── preferences.json          # Долгосрочная пользовательская память
    ├── capsules.json             # Атомарные заметки и капсулы комьюнити
    ├── events.jsonl              # Журнал событий (дистиллируется при 100)
    ├── goals.json                # Ваши жизненные цели (заполните под себя!)
    └── evolution_history.jsonl   # Журнал мутаций (для отката)
```
